Menús

Menú Lista

Crear lista - Crea una nueva lista y la hace activa.

Abrir lista - Abre una lista existente y la hace activa.

Guardar lista como... - Guarda la lista actual bajo otro nombre.

Importar - Use esta función para importar listas creadas por versiones viejas de %PROGRAMNAME%.

Refrescar lista - Refresca la lista actual

Estilo de la lista: Seleccione el estilo deseado para la lista de tareas.

Menú Tareas

Respaldar todo ahora - Respalda todas las tareas inmediatamente. Si se selecciona "Apagar el ordenador" en el diálogo de confirmación, su ordenador será apagado al concluir el respaldo.(Puede necesitar derechos de administrador)

Respaldar las tareas seleccionadas- Lo mismo que la función anterior, pero se respaldarán solamente las tareas seleccionadas.

Adicionar tarea - Adicionar una nueva tarea a la lista. Lea mas sobre las tareas aquí.

Modificar tarea - Abre la tarea seleccionada para modificar todas sus propiedades.

Clonar tarea - Crea una copia exacta de la tarea seleccionada. Esto puede ser útil cuando Vd. quiera crear una nueva tarea con los mismos ficheros y propiedades de otra ya existente.

Borrar tarea - Borra las tareas seleccionadas. Si Vd. selecciona "Borrar también los ficheros de respaldo" en el diálogo de confirmación, todos los ficheros de respaldo de esta tarea serán borrados. Aviso: esta operación es irreversible.

Abortar - Si un respaldo está llevándose a cabo, éste será abortado. Algunos ficheros temporales pueden quedarse sin borrar en su ordenador.

Restaurar los atributos de archivo: %PROGRAMNAME% utiliza el bit de Archivo de los atributos de un fichero para determinar si este debe ser copiado o no en caso de realizar un respaldo incremental o diferencial. Si Vd. quiere que el próximo respaldo de una tarea sea obligatoriamente de tipo respaldo completo, seleccione la tarea y escoja esta opción.

Apagar el ordenador: Si Vd. está realizando un respaldo MANUAL, Vd. puede ordenar que el ordenador se apague mientras el respaldo se está realizando, usando esta opción. Vd. puede también arrepentirse y retirar esta opción volviendo a hacer clic en este menú.

Menú Historia de respaldos

Borrar respaldo - Borra los respaldos seleccionados de la base de datos activa.  Si Vd. selecciona "Borrar también los ficheros de respaldo" en el diálogo de confirmación, todos los ficheros de respaldo de este respaldo serán borrados. Aviso: esta operación es irreversible.

Aparcado - Seleccione esta opción si quiere "aparcar" los respaldos seleccionados. Si un respaldo está aparcado, éste no podrá ser borrado automática- o manualmente.

Propiedades de respaldo: Muestra las propiedades del respaldo seleccionado.

Menú Diario de actividades o Log

%PROGRAMNAME% registra todas las operaciones del "motor" en un fichero de diario (log file). La interfaz de usuario registra también sus propias operaciones en la paleta "Eventos".

Vaciar la paleta de eventos - borra todo el texto de la paleta de "Eventos".

Seleccionar todo - Selecciona todo el texto de la paleta "Eventos".

Copiar el texto seleccionado -Copia el texto seleccionado

Imprimir - Imprime la paleta "Eventos".

Mostrar el diario de actividades - Muestra el diario de actividades del motor abriendo el fichero.

Borrar el diario de actividades - Borra el diario de actividades del motor

Herramientas

Actualizar el programa - Controla si hay alguna nueva versión del programa en el servidor. Esta tarea puede ser realizada automáticamente y se puede configurar en el diálogo de Opciones.

Herramientas de Descifrado - Ejecuta la herramienta de Descifrado para descifrar ficheros o directorios.

Herramienta de Traducción - Ejecuta la herramienta de traducción. Con la ayuda de esta Vd. podrá traducir los 2 ficheros necesarios para ejecutar la aplicación en su idioma. Envíeme los dos ficheros resultantes si desea incluirlos en la distribución oficial. Es muy importante que las cadenas a traducir en el fichero cui mantengan la longitud original para no deformar la interfaz de usuarios.

Opciones - Configurar el programa. Lea más aquí.
