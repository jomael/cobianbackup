Applicatie versus Service

Een service is een speciale Windows applicatie die ook draait als er geen gebruikers zijn aangemeld. Een service kan draaien onder een speciaal account (LocalSystem account) of onder een ander bestaand account. Bij Windows NT, 2000, XP of 2003 wordt aanbevolen het programma als een service te draaien. Deze keuze wordt gemaakt tijdens de installatie van het programma. De manier van draaien kan ook worden aangepast via de Instellingen.

Als %PROGRAMNAME% draait als een service kan er wel een probleem ontstaan: Als een shared drive of folder of enige niet locale resource wordt benaderd, kan de service niet onder de Local System account draaien. Deze moet dan draaien onder een account met netwerktoegang (voorbeeld: de Administrator account). In het installatie programma kan gekozen worden om de service te draaien onder de Local System account of een ander account.

Ander verschil is het stoppen van een service of een applicatie. Het stoppen van een service kan via Windows Taakbeheer / Processen. De service kan ook worden gecontroleerd via de Instellingen.
